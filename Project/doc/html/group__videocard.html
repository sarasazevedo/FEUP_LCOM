◆ vbe_vertical_retrace()

int vbe_vertical_retrace ( uint16_t x,
uint16_t y )

Set the display start during vertical retrace.

This function sets the display start coordinates (x, y) during the vertical retrace interval. This is useful to avoid visual artifacts such as tearing by ensuring the buffer swap happens during the vertical blanking interval.

Parameters
xThe x coordinate of the display start.
yThe y coordinate of the display start.
Returns
0 on success, 1 otherwise.
